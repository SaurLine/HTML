# Repositorio Ejmplo 1 HTML CSS GIT
## Nombre: ola

# Paso 0
1.- Descargar git<br>
2.- Descargar VSC
3.- Crear directorio y abrir VSC
4.- Abrir terminal y probar los comandos:
    git -v para ver la version del git
    git config --global user.name "nombre de usuario" | agregar usuario
    git config --global user.email "correo" | agregar correo del usuario
5.- Clonar el repositorio creado en GitHub:
    git colne "link del repositorio"

# Paso 1
Cada cambio que hacemos dentro del proyecto local para guardolo hacemos lo siguiente:
    git add . | preparar los archivos
    git commit -m "comentario del avance" | crear punto de guardado
    git push origin main | subir los cambios al repositorio
