# Repositorio Ejmplo 1 HTML CSS GIT
## Nombre: ola

# Paso 0
1.- Descargar git<br>
2.- Descargar VSC<br>
3.- Crear directorio y abrir VSC<br>
4.- Abrir terminal y probar los comandos:<br>
    git -v para ver la version del git<br>
    git config --global user.name "nombre de usuario" | agregar usuario<br>
    git config --global user.email "correo" | agregar correo del usuario<br>
5.- Clonar el repositorio creado en GitHub:<br>
    git colne "link del repositorio"<br>

# Paso 1<br>
Cada cambio que hacemos dentro del proyecto local para guardolo hacemos lo siguiente:<br>
    git add . | preparar los archivos<br>
    git commit -m "comentario del avance" | crear punto de guardado<br>
    git push origin main | subir los cambios al repositorio
